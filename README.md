# Sistema de Gimnasio

## Versión
**1.0**

## Descripción
Sistema de gestión de gimnasio desarrollado en Java. Permite gestionar miembros, membresías, clases grupales y control de acceso.

## Alcance

### ✅ Qué hace
- Registro y autenticación de miembros
- Gestión de membresías (Mensual, Trimestral, Anual, Vitalicia)
- Creación y listado de clases grupales (Yoga, Spinning, Crossfit, Pilates)
- Inscripción de miembros a clases
- Control de acceso (registro de ingresos y salidas)
- Estadísticas de uso (ocupación promedio, horas pico)
- Envío de notificaciones y recordatorios

### ❌ Qué NO hace
- No tiene interfaz gráfica (solo consola)
- No persiste datos en base de datos real
- No procesa pagos reales
- No envía notificaciones reales (solo simulación)
